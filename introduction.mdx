# Glober AI API Docs

Welcome to the official documentation for the Glober Platform.

We offer two core APIs:

---

## 🔐 Auth API
Handles:
- User registration
- Login/token generation
- Token verification
- Fetching user info

---

## 🧠 Project API
Manage translation projects:
- Create and configure projects
- Upload translation assets
- Track and retrieve outcomes

---
